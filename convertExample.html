This is exercise for creating a weight metric converter

We will take in a number in pounds(lbs)and convert it to kilogram(kg). Right now we only know enough to this one way, but later on we will learn how to add logic to this sort of problem!

To complete this exercise do the following

  • Link this HTML file to own .js file
  • Use Javascript to accept a number input in pounds
  • Then in your js file convert this number to kilogram(*0.454)
  • Afterwards report back in an alert what the weight is in kg
  • Then log/write "Conversion Completed" to the console
